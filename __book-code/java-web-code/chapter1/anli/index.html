窗内网简介
     窗内网（www.itzcn.com）是汇智科技继计算机教程网之后，又最新推出的在线教育平台。这是一个基于Web 2.0的真实社区，旨在推进无纸化教学进程，打造一个免费开放的网络学院。为入门级学员提供专家训练营系列课程，帮助学员系统的掌握流行技术；为提高级学员提供专题课程、精品讲座和企业学院系列课程，帮助快速学习新技术。
