## SpringBoot Redis演示项目

SpringBoot中通过RedisTemplate来直接操作Redis缓存数据库

## 运行

初始化sql文件在`resources/sql/t_user.sql`中

另外还需要安装Redis，配置好`application.yml`文件中的redis地址

测试用例：`com.xncoding.service.UserServiceTest.java`
